This FiveM ESX script adds a customizable receptionist NPC for jobs like police, ambulance, or fire department. It provides interactive options for players, enhances roleplay immersion, and streamlines job-related actions through a clean, easy-to-use interface
-------------------
Requirements
ox_lib (required)
ox_target (required)
Framework: ESX 
------------------
Installation
Copy the chooky_reception folder to resources.
Make sure ox_lib is started before this resource.
Start the resource in your server.cfg
